polyup

A helpful assistant for migrating from Polymer 0.5 to 1.0.
Does many of the boring mechanical parts for you.

View full docs and more on GitHub

The change from Polymer 0.5 to 1.0 is a large one, as we transitioned from exploratory beta releases to a stable production-ready product.

Fortunately polyup will parse your HTML and any javascript in either inline or external scripts and automatically take care of many of the steps involved in upgrading to Polymer 1.0!

Install it with npm install polyup
